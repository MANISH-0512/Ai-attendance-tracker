# Ai-attendance-tracker

# Link :-https://transparent-smart-campus-pulse.base44.app/

# Executive Summary :-

1.This project is a smart Attendance Tracker web application developed using the Base44 platform.

2.It allows teachers to mark and upload attendance, which students can view in real-time.

3.The system uses AI to assist in adding student names and managing class data efficiently.

4.It also automatically generates a detained list based on attendance criteria.

5.The project demonstrates automation of traditional attendance systems using AI.

# Business Problem :-

1.Traditional attendance systems are manual and prone to errors.

2.Teachers spend time marking attendance and calculating percentages.

3.Students often lack visibility into their attendance status.

4.Generating detained lists manually is time-consuming and inefficient.

5.This project aims to automate attendance tracking and improve transparency.

# Methodology :-

1.The application is built using the Base44 platform with prompt-based development.

2.Teachers can mark attendance and manage student lists with AI assistance.

3.Attendance data is processed to calculate percentages automatically.

4.A detained list is generated based on predefined attendance rules.

5.The system ensures real-time updates for both teachers and students.

# Skills & Tools Used :-

1.Tool Used: Base44 platform for web application development.

2.Skills: Prompt engineering, data handling, and system design.

3.Concepts: Attendance automation, AI-assisted data entry, and role-based access.

4.UI/UX: Dashboard design for teacher and student panels.

5.Focus on building a practical academic solution.

# Result :-

1.A functional attendance tracking system was successfully developed.

2.Teachers can easily mark attendance and manage student data.

3.Students can view their attendance status in real-time.

4.The system automatically generates detained lists accurately.

5.The project improves efficiency and reduces manual workload.
